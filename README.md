# VBC Simulation

Interactive simulation exploring ACO financial trade-offs under value-based care contracts.

**[Live Demo](https://pcplens.github.io/vbc-simulation/)**

## What It Covers

- **Funding models:** Bank Loan, Hospital Partnership, Payer Advance, Private Equity
- **Multi-year projections** with benchmark ratcheting, RAF tracking, and quality gate dynamics
- **Monte Carlo analysis** with Quasi-Monte Carlo (Sobol) sampling for robust scenario modeling
